# RO Tactical Board — Feature List

> กระดานวางแผนก่อนลงสนาม GW เปรียบเหมือน Tactical Board ของโค้ชฟุตบอล

---

## Core Features (สำคัญที่สุด)

### 1. Arrow Tool — ลูกศรแสดงเส้นทาง
- คลิก 2 จุดบนแผนที่เพื่อลากลูกศร
- บอกทิศทางการเคลื่อนที่ของแต่ละปาตี้ได้ชัดเจนกว่าเส้นฟรีแฮนด์
- เลือกสีลูกศรตามปาตี้ได้

### 2. บันทึกหลายแผน (Formations)
- Save ตำแหน่งโทเค็น + เส้นวาด เป็นชื่อแผน เช่น "แผน A — Offense", "แผน B — Defense"
- โหลดสลับแผนได้ทันที
- Admin เป็นคนจัดการ, Viewer เห็นแผนที่ถูกเลือกอยู่

### 3. Text Label บนแผนที่
- คลิกวางข้อความบนจุดใดก็ได้บนแผนที่
- ใช้บอก role, timing, หรือ instruction เช่น "Hold here", "Rush เมื่อสัญญาณ"

---

## ช่วยอธิบายแผน

### 4. Step-by-step (Phase การเล่น)
- แบ่งแผนเป็นขั้น 1 → 2 → 3
- กดถัดไปแล้วโทเค็นขยับตามลำดับ
- เหมือนโค้ชอธิบายทีละ phase ให้ทีมดูตาม

### 5. Notes ต่อปาตี้
- แต่ละปาตี้มีช่อง notes ของตัวเอง
- ใช้บอก role เฉพาะของปาตี้นั้น เช่น "ปาตี้ 3 — คอย Push ด้านข้าง อย่า Rush ก่อน"

---

## คุณภาพชีวิต (QoL)

### 6. Undo (Ctrl+Z)
- ยกเลิกการวาดทีละครั้ง
- ตอนนี้ถ้าวาดพลาดต้องล้างทั้งหมด

### 7. Export เป็นรูปภาพ
- ถ่าย screenshot ของแผนที่ทั้งหมดออกมาเป็น .png
- ส่งต่อให้ทีมผ่าน Discord / Line ได้ทันที

---

## ลำดับที่แนะนำให้สร้าง

| ลำดับ | ฟีเจอร์ | เหตุผล |
|---|---|---|
| 1 | Arrow Tool | สำคัญที่สุด — แผนจะสื่อสารออกมาได้จริง |
| 2 | Export เป็นรูป | ส่งต่อทีมได้ทันที ไม่ต้อง screenshot เอง |
| 3 | Text Label | เพิ่ม context บนแผนที่ |
| 4 | บันทึกหลาย Formation | รองรับหลายสถานการณ์ |
| 5 | Step-by-step | อธิบายแผนได้ละเอียดขึ้น |
| 6 | Notes ต่อปาตี้ | เสริม notes ที่มีอยู่แล้ว |
| 7 | Undo | polish สุดท้าย |
